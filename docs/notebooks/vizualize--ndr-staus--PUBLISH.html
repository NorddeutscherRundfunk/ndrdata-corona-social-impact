Auswertung NDR Staudaten¶

Anhand von Daten der vergangenen 4 Wochen soll berprft werden, wie sich die Einschrnkungen des ffentlichen Lebens auf das Stau-Aufkommen auswirken. Die Daten stammen aus dem Archiv der NDR-Verkehrsmeldungen.

Vorarbeit

Siehe Notebook process--ndr-staus.ipynb

Als Indikator nehmen wir die angegebene Staulnfe in km, via regex aus der Spalte attributes herausgelesen.
